**Q: Why Polymarket?**
A: Real markets, real liquidity, real resolutions based on realityâ€”not synthetic data or human judgment.

**Q: How prevent cheating?**
A: Models can't cheat reality. Markets resolve after predictions are made. Complete audit trail.

**Q: What about costs?**
A: Research project, not production service. Costs tracked transparently. Goal is honest evaluation.

**Q: When will results be ready?**
A: Markets resolve on their own schedules (days to months). Real-time updates as they resolve.

**Q: Can I add my model?**
A: If available via OpenRouter API, yes! Open source codebase for custom versions.

**Q: How is this different?**
A: Tests future events (can't be in training data) vs static datasets. Genuine reasoning vs memorization.
